# 🧪 Lab 3 – String Manipulation

## 📌 Overview

This lab focuses on string-based problems in Python. It covers important concepts such as string traversal, character frequency, palindrome checking, and basic data processing techniques.

---

## 🔤 Questions & Solutions

### ✅ Q1. Anagram Check

📄 

**Problem:**
Check whether two given strings are anagrams of each other.

**Approach:**

* Remove spaces and convert to lowercase
* Sort both strings
* Compare sorted results

**Example:**
Input: `"listen"`, `"silent"`
Output: `Strings are Anagrams`

---

### ✅ Q2. Character Frequency

📄 

**Problem:**
Count frequency of each character in a string.

**Approach:**

* Traverse string
* Store counts using a dictionary
* Print character-frequency pairs

**Example:**
Input: `"aabbc"`
Output:

```
a 2  
b 2  
c 1  
```

---

### ✅ Q3. String Palindrome

📄 

**Problem:**
Check whether a string is a palindrome.

**Approach:**

* Convert string to lowercase
* Compare string with its reverse (`[::-1]`)

**Example:**
Input: `"madam"`
Output: `Palindrome`

---

### ✅ Q4. Remove Duplicate Characters

📄 

**Problem:**
Remove duplicate characters while preserving order.

**Approach:**

* Traverse string
* Add characters only if not already present

**Example:**
Input: `"programming"`
Output: `"progamin"`
