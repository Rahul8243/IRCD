# 🧪 Lab 9 – Greedy Algorithms

## 📌 Overview

This lab focuses on **Greedy Algorithms**, where locally optimal choices lead to globally optimal solutions. These problems are common in scheduling, optimization, and compression techniques.

---

## 🔢 Questions & Solutions

### ✅ Q1. Activity Selection

📄 

**Problem:**
Select maximum number of non-overlapping activities.

**Approach:**

* Sort by finishing time
* Select activities greedily

---

### ✅ Q2. Fractional Knapsack

📄 

**Problem:**
Maximize value within given capacity.

**Approach:**

* Sort items by value/weight ratio
* Pick highest ratio first

---

### ✅ Q3. Huffman Coding

📄 

**Problem:**
Minimize total encoding length.

**Approach:**

* Build min heap
* Merge lowest frequency nodes
* Construct Huffman Tree

---

### ✅ Q4. Minimum Arrows to Burst Balloons

📄 

**Problem:**
Find minimum arrows to burst balloons.

**Approach:**

* Sort intervals by end
* Greedily shoot arrows

---

### ✅ Q5. Minimum Platforms Required

📄 

**Problem:**
Find minimum platforms needed for trains.

**Approach:**

* Sort arrival & departure
* Use two pointers
