# 🧪 Lab 8 – Heap & Priority Queue

## 📌 Overview

This lab focuses on **Heap (Priority Queue)** based problems. Heaps are efficient data structures used for selection problems, sorting, and frequency-based computations.

---

## 🔢 Questions & Solutions

### ✅ Q1. K Largest Elements

📄 

**Problem:**
Find the k largest elements in an array.

**Approach:**

* Use Min Heap of size k
* Compare elements and maintain heap

---

### ✅ Q2. Heap Sort

📄 

**Problem:**
Sort an array using Heap Sort.

**Approach:**

* Build Max Heap
* Swap root with last element
* Heapify repeatedly

---

### ✅ Q3. Top K Frequent Elements

📄 

**Problem:**
Find top k frequent elements in array.

**Approach:**

* Use Counter to count frequency
* Sort based on frequency
* Return top k elements

---

* Frequency counting
