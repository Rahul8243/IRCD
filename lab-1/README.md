# 🧪 Lab 1 – Basic Number Problems

---

## 🔢 Questions & Solutions

### ✅ Q1. Armstrong Numbers in Range

📄 

**Problem:**
Find all Armstrong numbers within a given range `[lower, upper]`.

**Approach:**

* Convert number to string to extract digits
* Raise each digit to power = number of digits
* Compare sum with original number

**Example:**
Input: `150 200`
Output: `153`

---

### ✅ Q2. Perfect Number

📄 

**Problem:**
Check whether a number is a Perfect Number.

**Approach:**

* Find all proper divisors (excluding number itself)
* Sum them
* Compare with original number

**Example:**
Input: `28`
Output: `Perfect Number`

---

### ✅ Q3. Palindrome Number

📄 

**Problem:**
Check whether a number is a palindrome.

**Approach:**

* Reverse the number using modulus and division
* Compare reversed number with original

**Example:**
Input: `121`
Output: `Palindrome`

---

### ✅ Q4. Count Prime Numbers

📄 

**Problem:**
Count the number of prime numbers in a given range.

**Approach:**

* Check divisibility up to √n
* Count valid prime numbers

**Example:**
Input: `10 20`
Output: `4`
